# Fluxos de trabalho comuns

Esta seção fornece documentação de referência detalhada para fluxos de trabalho comuns ao utilizar o Guardian Connector.

## O que você encontrará aqui

Documentação para realizar tarefas específicas com o Guardian Connector, como por exemplo:

- Fluxos de dados completos, desde uma ferramenta de coleta de dados até plataformas de visualização.
- Trabalhando com seus dados em ferramentas externas, como o QGIS.
- Exportar e fazer backup dos seus dados

Navegue pelos guias desta seção para encontrar fluxos de trabalho que atendam às suas necessidades.
