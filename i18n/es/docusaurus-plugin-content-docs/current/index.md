# Documentación de Guardian Connector

¡Bienvenido al centro de documentación de Guardian Connector! Esta guía completa te ayudará a comprender, implementar y usar Guardian Connector para satisfacer las necesidades de soberanía de datos y monitoreo ambiental de tu comunidad.

## Qué cubre esta documentación

Esta documentación proporciona una guía completa para:

### 🏗️ **[Visión General](./overview/)**
Comprender el propósito de Guardian Connector, sus principios de diseño y cómo encaja en el trabajo más amplio de monitoreo y defensa de tu comunidad.

### 📚 **[Referencia](./reference/)**
Información detallada sobre herramientas integradas, opciones de alojamiento y especificaciones técnicas para ayudarte a tomar decisiones informadas sobre la implementación.

### 🎓 **[Guías](./guides/)**
Guías completas para implementar las herramientas de Guardian Connector y establecer prácticas de datos centradas en la comunidad, incluyendo protocolos de soberanía de datos indígenas y flujos de trabajo específicos de la herramienta.

## Para quién es esta documentación

Esta documentación está diseñada para:

- **Líderes Comunitarios** que planifican y supervisan iniciativas de soberanía de datos
- **Investigadores de Campo** que recolectan datos ambientales y culturales
- **Gestores de Datos** que organizan y analizan la información comunitaria
- **Coordinadores Técnicos** que implementan y mantienen los sistemas de Guardian Connector
- **Defensores** que utilizan datos para campañas de conservación y basadas en derechos

## Primeros Pasos

Usa la barra de navegación superior para explorar cada sección de este sitio de documentación:

1. **Comienza con [Overview](./overview/)** para comprender el propósito y los principios de diseño de Guardian Connector
2. **Revisa [Reference](./reference/)** para conocer las herramientas integradas y las opciones de implementación  
3. **Sigue las [Guides and Tutorials](./guides/)** para un aprendizaje práctico y el desarrollo de habilidades
