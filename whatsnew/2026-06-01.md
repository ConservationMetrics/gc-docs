This month’s updates focus on improving the user experience, enhancing platform performance, and expanding data integration capabilities.

<!-- truncate -->

### **1. User Experience & Interface**

#### **Landing Page Redesign and Usability**

  * The login page has been redesigned with a new layout, updated logos, and improved language picker functionality.
  ![login page](/img/whatsnew/login.jpg)
  * **Improved User Guidance:** The landing page now provides clearer information on how to integrate data, including examples of compatible apps like CoMapeo and KoboToolbox, to assist new users.
  ![connected apps](/img/whatsnew/connected-apps.jpg)
  * **Accessibility:** Swahili and Thai language support have been added to the platform.

#### **Explorer Interface and Consistency**

  * **Visual Standardization:** The platform's visual identity is now more consistent, with and uniform color usage and icon styling.
  * **Enhanced Feedback:** Empty state screens for various views, such as the Gallery view, have been introduced to provide better feedback when no data is available, replacing previous blank screens.
  * **Alerts and Statistics:** The alerts feature now allows users to download statistics that are filtered by the currently selected date range.
    ![statistics download](/img/whatsnew/statistics download.jpg)

### **2. Data Handling and Connectors**

#### **Data Performance Improvements**

  * **Data Loading Limit:** To prevent performance issues when visualizing large datasets, a hardcoded limit of 10,000 records has been implemented for data visualization in GC Explorer.
    A toast notification now alerts users when this limit is triggered.
    ![partial results](/img/whatsnew/partial results.png)

#### **New External Connectors**

  * **Cyber Tracker Connector:** Support for [Cyber Tracker](https://www.cybertracker.org/) has been added to the [Windmill Script Hub](https://github.com/ConservationMetrics/gc-scripts-hub/tree/main/f/connectors/cybertracker).
  * **GeoPackage Uploads:** Support for [GeoPackage](https://www.geopackage.org/) file uploads has been added to accommodate workflows from QGIS and [QField](https://docs.qfield.org/get-started/tutorials/advanced-setup-qfc/#working-modes).
  
### **3. Engineering and Documentation**

  * **Deployment Fixes:** Deployment documentation has been improved to address common issues encountered during the installation process.
  * **Documentation:** A new guide for [Local Contexts](https://localcontexts.org/) has been added to the documentation, providing instructions on project setup, label curation, and resource configuration within the Windmill tool.
