# 🔊 + 🦉 Guia: Fundamentos da Bioacústica com Sensores AudioMoth 

Este guia fornece uma visão geral do fluxo de trabalho dos sensores AudioMoth. Estes materiais apresentam aos usuários todo o ciclo de vida da implantação, manutenção e gerenciamento de dados provenientes dos sensores bioacústicos AudioMoth, conforme sugerido pela [Conservation Metrics (CMI)](https://conservationmetrics.com).

Os sensores AudioMoth são dispositivos de gravação acústica de baixo custo e de código aberto, amplamente utilizados no monitoramento ecológico. Este guia apresenta as melhores práticas para preparar os dispositivos, configurar o firmware e definir os horários de gravação, realizar instalações em campo, gerenciar a reinstalação e processar os dados coletados. O objetivo é garantir a coleta consistente e confiável de dados em diferentes projetos e equipes.
