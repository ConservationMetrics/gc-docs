# Modelos disponíveis

Esta página fornece links para modelos XLSForm projetados para objetivos comuns de coleta de dados por organizações de comunidades indígenas. Esses modelos podem ser usados no KoboToolbox, ODK Central ou em qualquer outra plataforma compatível com XLSForm.

:::informações Biblioteca em Expansão
Esta é uma biblioteca crescente de modelos e estamos abertos a contribuições! Se você tem modelos que seriam úteis para comunidades indígenas, por favor, considere compartilhá-los conosco enviando uma solicitação de pull request ou entrando em contato conosco.

## Modelos disponíveis

### 📷 Implantação de Câmeras de Armadilha

Um formulário básico para registrar informações sobre a instalação de armadilhas fotográficas no campo.

**Caso de uso**: Registrar os locais onde as câmeras fotográficas são instaladas, os detalhes da configuração e as condições de implantação para projetos de monitoramento da vida selvagem.

**[Baixar Modelo](/xlsform-templates/configuração_de_armadilhas_fotográficas_(básico).xlsx)**

### 💬 Histórias orais baseadas em locais

Um formulário abrangente para coletar histórias orais relacionadas a locais específicos, com foco na coleta de narrativas, no registro dos lugares mencionados nessas histórias e no registro de informações sobre os contadores de histórias.

**Caso de uso:** Criar conjuntos de dados para o [Terrastories](https://terrastories.app), uma plataforma para mapeamento e compartilhamento de histórias relacionadas a locais específicos. Este modelo ajuda as comunidades a coletar e organizar sistematicamente narrativas orais com contexto geográfico.

**[Baixar Modelo](/xlsform-templates/histórias_orais_locais.xlsx)**

### 🌿 Monitorização biocultural

Um exemplo de formulário com perguntas projetadas para o monitoramento biocultural, medindo o bem-estar a partir da perspectiva dos povos indígenas.

**Caso de uso:** Monitorar as relações interligadas entre a diversidade biológica e as práticas culturais, avaliando o bem-estar da comunidade através de uma perspectiva indígena que reconhece a conexão inseparável entre as pessoas e o local onde vivem.

**[Baixar Modelo](/xlsform-templates/biocultural_monitoring.xlsx)**

## Como utilizar esses modelos

1. **Baixe** o arquivo XLS para o modelo escolhido.

2. **Escolha a sua abordagem de personalização:**

   **Opção A: Editar no Criador de Formulários do KoboToolbox**
   - Faça o upload do modelo para o KoboToolbox diretamente.
   - Utilize o construtor de formulário visual para revisar e personalizar perguntas, lógica e configurações.
   - Implemente quando estiver pronto.

   **Opção B: Editar o arquivo XLS diretamente**  
   - Revise e personalize o arquivo XLS offline (perguntas, lógica, traduções, etc.).
   - Envie seu arquivo XLS personalizado para o KoboToolbox ou para a plataforma de sua preferência.
   - Implemente quando estiver pronto.

3. **Teste** o formulário completamente antes de utilizá-lo para coletar dados.

4. **Treine** seus coletores de dados sobre a estrutura do formulário e os objetivos.

## Diretrizes de Personalização

- **Idioma:** Adicione traduções para idiomas locais conforme necessário
- **Perguntas:** Modifique, adicione ou remova perguntas com base em seus objetivos específicos.
- **Lógica:** Ajuste a lógica de pulo e as regras de validação para corresponder ao seu fluxo de trabalho.
- **Mídia:** Defina as configurações apropriadas de coleta de mídia para o seu contexto.
- **Consulte recursos online:**
  - [Documentação do KoboToolbox](https://support.kobotoolbox.org/)
  - [Documentação do ODK](https://docs.getodk.org/form-design-intro/)
  - [Documentação do XLSForm](https://xlsform.org/)
  - [Canal do YouTube da Humanitarian Data Solutions](https://www.youtube.com/@HumanitarianDataSolutions)

## Modelos para contribuir

Se você desenvolveu modelos XLSForm que seriam úteis para outras comunidades indígenas, nós o encorajamos a compartilhá-los! Os modelos devem ser:

- **Comunidade no centro:** Projetado com foco nas necessidades e perspectivas das comunidades indígenas.
- **Bem documentado**: Inclua descrições claras de casos de uso e objetivos.
- **Testado**: Foi utilizado com sucesso em cenários reais de coleta de dados.
- **Adequado do ponto de vista cultural:** Respeitoso com os sistemas e protocolos de conhecimento das comunidades indígenas.

Entre em contato conosco para discutir a inclusão dos seus modelos nesta biblioteca.
