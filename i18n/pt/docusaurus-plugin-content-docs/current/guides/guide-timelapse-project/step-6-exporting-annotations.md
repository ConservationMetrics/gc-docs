# Passo 6: Exportação das Anotações

## Exportar como CSV

Todos os dados gerados durante a revisão são armazenados localmente no seu computador. Para exportar:

- Utilize a função no menu **Arquivo → Exportar ou importar dados para/de um arquivo CSV** dentro do Timelapse.
- Você pode optar por exportar uma seleção atual dos seus dados, ou exportar todos os dados.
- Agora suas anotações estão disponíveis como um arquivo CSV para análises adicionais. Ou, você pode importar os dados novamente para um projeto Timelapse.
