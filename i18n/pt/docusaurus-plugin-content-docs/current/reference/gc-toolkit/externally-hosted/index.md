# Serviços Hospedados Externamente

O Guardian Connector atualmente depende de serviços de terceiros (que não são hospedados em uma instância do Guardian Connector) para fornecer funcionalidades adicionais:

- [Auth0](./auth0/) - Plataforma de autenticação e autorização
- [Mapbox](./mapbox/) - Plataforma para mapeamento e visualização de dados geográficos
- [Twilio](./twilio/) – Plataforma para comunicação por SMS e voz
- [Uptime Robot](./uptime-robot/) - Monitoramento de sites e acompanhamento da disponibilidade
