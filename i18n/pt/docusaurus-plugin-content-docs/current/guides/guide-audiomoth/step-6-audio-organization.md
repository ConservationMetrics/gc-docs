# Passo 6: Transferência e Gerenciamento de Dados

Os arquivos devem ser organizados de forma semelhante à documentação disponível em [dados de armadilhas fotográficas](`/guias/guia-projeto-time-lapse/passo-1-organizando-imagens.md).

## Copiar Dados

0. Crie uma pasta para recuperação de dados. O nome desta pasta deve seguir as seguintes convenções: ``{organization_initials}_{project_name}_{project_year}_{R#}``, onde ``organization_initials`` é um nome abreviado da organização que coleta/possui os dados, ``project_name`` é uma descrição de 1 a 3 palavras do projeto, ``project_year`` é o ano em que os dados foram coletados e ``R#`` é o número do evento (ou "rodada") de coleta de dados. Assim, na primeira vez que você coletar as cartões SD, você usaria R1, e na segunda vez, usaria R2, etc.

> Ex. Fazenda Wilder, CMI\_2025\_Versão 1

1.  Após recuperar os dados, copie cada cartão SD para uma **pasta com o nome especificado**.
    ID do cartão\*. Esta pasta deve ser colocada na pasta de recuperação de dados que você criou no passo anterior.

2.  Armazene essas pastas no computador do seu projeto.

## Planilhas de Campo

-   Mantenha os metadados de implantação e recuperação no Google acumulado.
    Folha.

-   Mantenha os dados acústicos no computador do projeto por **≥6 meses** como um
    cópia de segurança.

## Problemas com cartões SD

-   Cartões SD danificados ou ilegíveis podem ser enviados para a CMI para recuperação.

-   Cartões com problemas repetidos devem ser recolhidos/descartados.
