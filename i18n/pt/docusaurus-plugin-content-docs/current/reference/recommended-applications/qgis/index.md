# QGIS

**QGIS** é uma aplicação de código aberto poderosa para trabalhar com dados geográficos, como arquivos shape, pacotes Geo, dados coletados usando [CoMapeo](/reference/connected-applications/comapeo), e outros formatos de dados espaciais. Ele permite que você crie, edite e visualize mapas de inúmeras maneiras. 

Você pode integrar o QGIS com o Guardian Connector de duas maneiras principais:

1. **Importação de dados** do Guardian Connector para o QGIS, com o objetivo de realizar análises ou visualizações espaciais.
2. **Exportar dados processados** ou resultados de mapas do QGIS de volta para o Guardian Connector, a fim de enriquecer seus conjuntos de dados com informações geográficas.

Isso torna o QGIS uma ferramenta flexível e amigável para a comunidade, ideal para qualquer pessoa que trabalhe com informações de terra, território e dados espaciais.
