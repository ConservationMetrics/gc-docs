# Para desenvolvedores

O Guardian Connector é um ecossistema de código aberto composto por ferramentas interconectadas, projetado para apoiar a soberania de dados das comunidades indígenas e o monitoramento ambiental baseado na comunidade. Para desenvolvedores interessados em contribuir, implantar ou estender o Guardian Connector, esses repositórios fornecem os elementos básicos e a infraestrutura de implantação necessários.

## 📦 Repositórios Principais

- **[Implantação do Guardian Connector](https://github.com/ConservationMetrics/gc-deploy)** - Guias completos de implantação e configuração da infraestrutura para a pilha Guardian Connector em ambientes na nuvem ou locais.
- **[Hub de Scripts do Guardian Connector](https://github.com/ConservationMetrics/gc-scripts-hub)** – Coleção de scripts, fluxos de trabalho e aplicações alimentadas por vento, que apoiam os gestores de terras em todo o mundo.
- **[Explorer do Guardian Connector](https://github.com/ConservationMetrics/gc-explorer)** - Aplicação web de front-end para visualização unificada de dados espaciais e tabulares da comunidade, em diferentes vistas: mapa, galeria e painel de controle

## 💡 Filosofia de Desenvolvimento

O desenvolvimento do Guardian Connector é guiado pelos seguintes princípios:

- **Soberania dos Dados Indígenas:** As comunidades mantêm o controle total sobre seus dados e infraestrutura.
- **Colaboração de Código Aberto**: Todos os componentes estão disponíveis gratuitamente e incentivam a contribuição da comunidade.
- **Prioridade para a Acessibilidade**: As ferramentas devem ser fáceis de usar pelos membros da comunidade, sem exigir conhecimentos técnicos avançados.
- **Arquitetura Modular:** Os componentes funcionam de forma independente, mas se integram perfeitamente.
- **Design Centrado na Comunidade:** As características priorizam as necessidades da comunidade em vez da conveniência técnica.

## 🤝 Colaborando

O Guardian Connector recebe contribuições de desenvolvedores que compartilham nosso compromisso com a soberania de dados das comunidades indígenas e o monitoramento ambiental controlado pela comunidade. Seja corrigindo bugs, adicionando funcionalidades, melhorando a documentação ou propondo novas integrações, suas contribuições ajudam a fortalecer comunidades em todo o mundo.

Cada repositório contém diretrizes específicas de contribuição, mas todas as contribuições devem estar em consonância com nossos princípios fundamentais de acessibilidade, controle da comunidade e soberania dos dados.
