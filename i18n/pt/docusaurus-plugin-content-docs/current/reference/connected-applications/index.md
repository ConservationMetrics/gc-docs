# Aplicativos Conectados

O Guardian Connector atua como um centro central para coletar e agregar dados de várias fontes. Ao se conectar a diversas ferramentas e plataformas de coleta de dados, o Guardian Connector consolida os dados gerados pela comunidade em um único banco de dados, facilitando o gerenciamento e o acesso.

## Ferramentas de Coleta de Dados Essenciais

- **[KoboToolbox](./kobotoolbox/)** – Coleta de dados móvel e gerenciamento de pesquisas
- **[CoMapeo](./comapeo/)** - Mapeamento e coleta de dados offline, com foco nas comunidades.
- **[Vídeo em velocidade acelerada](./timelapse/)** – Análise de imagens de armadilhas fotográficas e monitoramento da vida selvagem
  
Essas ferramentas fornecem acesso direto aos dados para o Guardian Connector, permitindo a sincronização automática de dados coletados no campo com o banco de dados central.

## Outras fontes de dados

O Guardian Connector pode importar e integrar dados de várias outras plataformas externas, que são comumente utilizadas por comunidades indígenas e organizações de conservação:

- **[ArcGIS Online](https://www.arcgis.com/)** – Plataforma completa de SIG baseada na nuvem da Esri para gerenciamento e análise de dados geoespaciais.
- **[Global Forest Watch](https://www.globalforestwatch.org/)** – Sistema de monitoramento florestal em tempo real, que fornece alertas sobre desmatamento e imagens de satélite.
- **[Mapa Locus](https://www.locusmap.app/)** - Aplicativo móvel de mapeamento para navegação ao ar livre e coleta de dados GPS.
- **[ODK](https://getodk.org/)** - Plataforma móvel de coleta de dados de código aberto para criar e implantar pesquisas. (Leia sobre as diferenças entre KoboToolbox e ODK [aqui](/reference/connected-applications/kobotoolbox/#kobotoolbox-or-odk).)
- **[OpenStreetMap](https://www.openstreetmap.org/)** – Projeto de mapeamento colaborativo, impulsionado pela comunidade, que cria dados geográficos gratuitos.
- **[QGIS](https://qgis.org/)** – Aplicação de SIG (Sistema de Informação Geográfica) de código aberto para uso em desktops, ideal para análises espaciais avançadas e produção de mapas.

Essas integrações de fontes de dados permitem que o Guardian Connector agregue informações de diversos métodos e plataformas de coleta, criando um repositório abrangente de dados da comunidade, ao mesmo tempo em que mantém a soberania dos dados e o controle local.
