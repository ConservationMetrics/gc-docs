# Documentation Guardian Connector

Bienvenue au centre de documentation Guardian Connector ! Ce guide complet vous aidera à comprendre, déployer et utiliser Guardian Connector pour répondre aux besoins de votre communauté en matière de souveraineté des données et de surveillance environnementale.

## Ce que couvre cette documentation

Cette documentation fournit des directives complètes pour :

### 🏗️ **[Vue d'ensemble](./overview/)**
Comprendre l'objectif et les principes de conception de Guardian Connector, et comment il s'intègre dans le travail plus large de surveillance et de plaidoyer de votre communauté.

### 📚 **[Référence](./reference/)**
Informations détaillées sur les outils intégrés, les options d'hébergement et les spécifications techniques pour vous aider à prendre des décisions éclairées concernant la mise en œuvre.

### 🎓 **[Guides](./guides/)**
Guides complets pour la mise en œuvre des outils Guardian Connector et l'établissement de pratiques de données centrées sur la communauté, y compris les protocoles de souveraineté des données autochtones et les flux de travail spécifiques aux outils.

## À qui s'adresse cette documentation

Cette documentation est conçue pour :

- **Leaders communautaires** planifiant et supervisant les initiatives de souveraineté des données
- **Chercheurs de terrain** collectant des données environnementales et culturelles
- **Gestionnaires de données** organisant et analysant les informations communautaires
- **Coordonnateurs techniques** mettant en œuvre et maintenant les systèmes Guardian Connector
- **Militants** utilisant les données pour la conservation et les campagnes basées sur les droits
