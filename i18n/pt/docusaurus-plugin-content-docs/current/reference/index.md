# Introduction

Esta seção de referência fornece informações técnicas detalhadas, especificações e orientações práticas para ajudá-lo a implementar, configurar e utilizar o Guardian Connector de forma eficaz. Seja você planejando uma implantação, integrando ferramentas ou gerenciando as operações diárias, encontrará as informações abrangentes de que precisa aqui.

## Seções de referência

### 📋 **[Kit de Ferramentas do Guardian Connector](./gc-toolkit/)**

Documentação abrangente para a plataforma Guardian Connector, incluindo o [Hub de Scripts GC](/referência/toolkit-gc/hub-de-scripts-gc/), o [Explorer GC](/referência/toolkit-gc/explorer-gc/) e muito mais.

**O que você encontrará:**
- Documentação de referência para o uso do GC Scripts Hub, GC Explorer, Apache Superset, Filebrowser e outras ferramentas.
- Como utilizamos serviços externos para fornecer funcionalidades adicionais ao Guardian Connector

**Ideal para:** Compreender a plataforma Guardian Connector, planejar fluxos de trabalho e aprender como diferentes ferramentas funcionam em conjunto no ecossistema Guardian Connector.

### 🔗 **[Aplicativos Conectados](./connected-applications/)**

Documentação para ferramentas e plataformas externas que o Guardian Connector pode integrar como um centro de dados centralizado.

**O que você encontrará:**
- Ferramentas de coleta de dados principais (KoboToolbox, CoMapeo, Timelapse) que são suportadas no Guardian Connector. 
- Plataformas externas com as quais o Guardian Connector pode integrar-se como um centro de dados centralizado.

**Ideal para:** Compreender os fluxos de trabalho de coleta de dados, planejar integrações e aprender como o Guardian Connector agrega dados de várias fontes.

### 🌐 **[Aplicações Recomendadas](./recommended-applications/)**

Documentação para aplicações recomendadas que são úteis para tarefas e fluxos de trabalho comuns.

**O que você encontrará:**
- Orientações sobre o uso de ferramentas que consideramos úteis, como [QGIS](/reference/recommended-applications/qgis) e [OpenRefine](/reference/recommended-applications/openrefine].

### 📚 **[Fluxos de Trabalho Comuns](./common-workflows/)**

Orientações para fluxos de trabalho comuns ao utilizar o Guardian Connector.

**O que você encontrará:**
- Orientações sobre como usar seus dados em [QGIS](referência/aplicações-recomendadas/qgis) ou em outro software SIG.
- Informações sobre como baixar todos os seus dados

### 🔧 **[Para Desenvolvedores](./para-desenvolvedores/)**

Informações para desenvolvedores interessados em contribuir para o Guardian Connector ou criar integrações personalizadas.

**O que você encontrará:**
- Links para os repositórios principais
- Configuração do ambiente de desenvolvimento
- Instruções de implantação e hospedagem
- Diretrizes para contribuições
