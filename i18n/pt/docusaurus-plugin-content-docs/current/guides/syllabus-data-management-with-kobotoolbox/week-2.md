# Semana 2: Gerenciamento de pesquisas e dados no KoboToolbox

## **Objetivos**

* Gerenciar o acesso dos usuários ao questionário.
* Exporte os dados coletados em diversos formatos.

## **📘 Introdução**

Uma gestão eficaz de pesquisas garante a integridade e o acesso aos dados. Nesta semana, o foco é na gestão de pesquisas, permissões de usuários e na exportação de dados para análise.

## **📚 Materiais de Aprendizagem**

* **Versão de Formulários:** [Implementando e Reimplementando Formulários no KoboToolbox](https://support.kobotoolbox.org/deploy_form_new_project.html)
* **Gerenciamento de Usuários:** [Gerenciar permissões no KoboToolbox](https://support.kobotoolbox.org/managing_permissions.html)
* **Tutorial em vídeo:** [Exportar dados do KoboToolbox para o Excel](https://www.youtube.com/watch?v=Nqsveh_UcRo)
* **Guia de Exportação de Dados:** [Exportar e Enviar Dados para Software SIG](https://support.kobotoolbox.org/upload_to_gis.html)

## **Atividades**

1. **Gerenciamento de Usuários:** Adicione e gerencie as permissões dos usuários para sua pesquisa. Pratique adicionando usuários e definindo diferentes níveis de acesso com outra pessoa do workshop.
2. **Exportação de Dados:** Exporte os dados coletados nos formatos CSV e Excel, e veja como eles aparecem na sua ferramenta de planilha favorita.
3. **Controle de Versões:** Atualize seu questionário, publique-o novamente e observe como o controle de versões funciona.
4. **Compartilhe sua experiência** Tire capturas de tela ou fotos da sua experiência e carregue-as em [**o questionário vinculado aqui**](https://ee.kobotoolbox.org/x/WXv1PxMx)

## **📚 Recursos adicionais obtidos durante o processo de check-in**

Estes são tópicos que foram discutidos na última reunião, para que todos possam estar informados e ter acesso aos materiais ou às gravações da sessão.

### **Agrupamento de perguntas e lógica de pulo**

As pessoas estavam interessadas em entender melhor como o agrupamento de perguntas funcionava e como isso se relacionava com a lógica de "pular" (skip logic).

Aqui está um recurso que explica os agrupamentos de perguntas: [https://support.kobotoolbox.org/group\_repeat.html](https://support.kobotoolbox.org/group_repeat.html).

### **Mecanismos de coleta de dados**

Existem duas formas principais de coletar dados: utilizando o aplicativo Kobo Collect para Android e utilizando formulários na web (também conhecidos como Enketo).

O KoboToolbox possui uma [documentação excelente que descreve detalhadamente cada um desses mecanismos](https://support.kobotoolbox.org/data-collection-tools.html).

### **Exportar para o SPSS**

Um dos funcionários mencionou familiaridade com o SPSS, uma ferramenta de estatística.

Este vídeo mostra como exportar dados do KoboToolbox para o SPSS para análise posterior: [Como importar dados do KoboToolbox para o SPSS para análise](https://www.youtube.com/watch?v=vqIvkqPhyeM)
