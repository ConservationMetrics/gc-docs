# Documentation Guardian Connector

Bienvenue sur le centre de documentation Guardian Connector ! Ce guide complet vous aidera à comprendre, déployer et utiliser Guardian Connector pour répondre aux besoins de votre communauté en matière de souveraineté des données et de surveillance environnementale.

## Ce que couvre cette documentation

Cette documentation fournit des conseils complets pour :

### 🏗️ **[Vue d'ensemble](./overview/)**
Comprendre l'objectif de Guardian Connector, ses principes de conception et comment il s'intègre dans le travail plus large de surveillance et de plaidoyer de votre communauté.

### 📚 **[Référence](./reference/)**
Informations détaillées sur les outils intégrés, les options d'hébergement et les spécifications techniques pour vous aider à prendre des décisions éclairées concernant la mise en œuvre.

### 🎓 **[Guides](./guides/)**
Guides complets pour la mise en œuvre des outils Guardian Connector et l'établissement de pratiques de données centrées sur la communauté, y compris les protocoles de souveraineté des données autochtones et les flux de travail spécifiques aux outils.

## À qui s'adresse cette documentation

Cette documentation est conçue pour :

- **Leaders communautaires** planifiant et supervisant les initiatives de souveraineté des données
- **Chercheurs sur le terrain** collectant des données environnementales et culturelles
- **Gestionnaires de données** organisant et analysant les informations communautaires
- **Coordinateurs techniques** implémentant et maintenant les systèmes Guardian Connector
- **Défenseurs** utilisant les données pour la conservation et les campagnes basées sur les droits

## Démarrer

Utilisez la barre de navigation supérieure pour explorer chaque section de ce site de documentation :

1. **Commencez par la [Vue d'ensemble](./overview/)** pour comprendre l'objectif et les principes de conception de Guardian Connector
2. **Consultez la [Référence](./reference/)** pour en savoir plus sur les outils intégrés et les options de mise en œuvre  
3. **Suivez les [Guides et Tutoriels](./guides/)** pour un apprentissage pratique et le développement de compétences
