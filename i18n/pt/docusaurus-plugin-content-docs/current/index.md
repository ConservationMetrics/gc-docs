# Documentação do Guardian Connector

Bem-vindo ao centro de documentação do Guardian Connector! Este guia completo o ajudará a entender, implementar e utilizar o Guardian Connector para atender às necessidades da sua comunidade em relação à soberania de dados e monitoramento ambiental.

## O que esta documentação cobre

Esta documentação oferece um guia abrangente para:

### 🚀 **[Visão geral](./overview/)**
Compreender o propósito do Guardian Connector, seus princípios de projeto e como ele se encaixa no trabalho mais amplo de monitoramento e defesa da sua comunidade.

### 📚 **[Referências](./reference/)**
Informações detalhadas sobre ferramentas integradas, opções de hospedagem e especificações técnicas para ajudá-lo a tomar decisões informadas sobre a implementação.

### 🎓 **Guias** (encontrados em `/guias/`)
Guias abrangentes para a implementação de ferramentas do Guardian Connector e para o estabelecimento de práticas de coleta de dados centradas na comunidade, incluindo protocolos de soberania de dados indígenas e fluxos de trabalho específicos para cada ferramenta.

## Para quem este documento se destina

Esta documentação foi elaborada para:

- **Líderes Comunitários** planejando e supervisionando iniciativas de soberania de dados
- **Pesquisadores de campo** coletando dados ambientais e culturais
- **Responsáveis pela gestão de dados**, que organizam e analisam informações da comunidade.
- **Coordenadores Técnicos** responsáveis pela implementação e manutenção dos sistemas Guardian Connector.
- **Defensores** que utilizam dados para campanhas de conservação e de defesa dos direitos

## Como começar

Utilize a barra de navegação principal para explorar cada seção deste site de documentação:

1. **Comece com a seção "[Visão geral]" (](./overview/)) para entender o propósito e os princípios de design do Guardian Connector.
2. **Revise o documento de referência](./reference/) para obter informações sobre ferramentas integradas e opções de implementação.  
3. **Consulte os guias e tutoriais](./guides/) para aprender de forma prática e desenvolver habilidades práticas.
