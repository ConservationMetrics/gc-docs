# Introduction

Bem-vindo à seção de tutoriais do Guardian Connector! Aqui você encontrará guias completos e passo a passo, projetados para ajudar organizações de comunidades indígenas a aproveitar ao máximo as ferramentas integradas do ecossistema do Guardian Connector.

## O que você encontrará aqui

Nossos tutoriais são elaborados levando em consideração as necessidades da comunidade, com foco em habilidades práticas que apoiam o monitoramento ambiental, a soberania de dados e a defesa da conservação. Seja você um iniciante na coleta de dados digitais ou busca expandir suas capacidades técnicas, estes guias oferecem instruções claras e acessíveis.

## Recursos de Aprendizagem Disponíveis

### 📚 [Currículo de Gestão de Dados com KoboToolbox](./syllabus-data-management-with-kobotoolbox/)

Um programa abrangente de aprendizado autogerenciado, com duração de 4 semanas, que o leva do básico do KoboToolbox às técnicas avançadas de gerenciamento de dados. Perfeito para comunidades que desejam estabelecer fluxos de trabalho robustos de coleta de dados móveis.

**O que você aprenderá:**
- Criar e implementar questionários para coleta de dados de campo
- Gerenciamento da qualidade e validação de dados
- Design avançado de formulários com lógica condicional e cálculos.
- Estratégias de análise e exportação de dados
- Fluxos de integração com outras ferramentas do Guardian Connector.

**Formato:** Módulos semanais com tutoriais em vídeo, materiais de leitura e exercícios práticos.

### 🐾 [Guia de Configuração do Projeto Timelapse](./guide-timelapse-project/)

Um guia detalhado e prático para configurar projetos de análise de dados de armadilhas fotográficas utilizando o software Timelapse. Projetado para ajudar as comunidades a documentar a presença da vida selvagem e a biodiversidade em seus territórios.

**O que você aprenderá:**
- Organização de imagens capturadas por armadilhas fotográficas para análise eficiente
- Criar modelos de anotação personalizados com nomes de espécies específicos da comunidade
- Gerenciamento de metadados e organização de pastas
- Fluxo de trabalho para revisão e etiquetagem de imagens
- Exportar dados para análise e elaboração de relatórios adicionais.

**Formato:** Tutorial passo a passo com capturas de tela, exemplos práticos e recursos para download.

### 📜 Protocolos de Compartilhamento de Dados Baseados na Soberania de Dados Indígenas (ver guia: soberania de dados e protocolos)

Um guia completo para criar protocolos de compartilhamento de dados baseados nos princípios de soberania de dados dos povos indígenas, projetado para comunidades que desejam estabelecer estruturas éticas e de governança de dados controladas pela comunidade.

**O que você aprenderá:**
- Compreendendo a Soberania de Dados das Populações Indígenas e os Princípios do CARE
- Estabelecer protocolos de compartilhamento de dados controlados pela comunidade
- Implementando estruturas de Consentimento Livre, Precoce e Informado (CPI)
- Elaboração de Acordos de Entendimento para parcerias de dados
- Proteger informações culturais e territoriais sensíveis
- Desenvolver a capacidade de governança de dados na comunidade

### 🏷️ [Rotulagem de Conjuntos de Dados com Rótulos de Contexto Local](./guia-contextos-locais/)

Um fluxo de trabalho prático para comunidades que desejam usar as etiquetas [Local Contexts](https://localcontexts.org/) e BC com o Guardian Connector: configurar um projeto central, organizar as etiquetas, sincronizá-las com o Windmill e aplicar as etiquetas escolhidas a conjuntos de dados específicos sem alterar a tabela de dados subjacente.

**O que você aprenderá:**
- Criar uma comunidade para o projeto "Hub de Contextos Locais" e organizar os rótulos TK/BC.
- Configurar os recursos do Windmill e executar o módulo "Local Contexts: Fetch Labels
- Utilizando o aplicativo "**GC Local Contexts Annotations**" para mapear rótulos para tabelas de armazém
- Como os metadados de etiquetas por conjunto de dados (``__lc_labels``) são armazenados separadamente das linhas do conjunto de dados.

**Formato:** Guia com várias páginas (projeto Hub e etiquetas, script de obtenção de etiquetas, aplicativo de anotações), além de links para a documentação oficial da Local Contexts.
