# Introduction

Esta sección de referencia proporciona información técnica detallada, especificaciones y orientación práctica para ayudarle a implementar, configurar y utilizar Guardian Connector de forma eficaz. Ya sea que esté planificando una implementación, integrando herramientas o gestionando operaciones diarias, aquí encontrará la información completa que necesita.

## Secciones de referencia

### 📋 **[Kit de herramientas de Guardian Connector](./gc-toolkit/)**

Documentación completa para la plataforma Guardian Connector, incluyendo el [GC Scripts Hub](/reference/gc-toolkit/gc-scripts-hub/), [GC Explorer](/reference/gc-toolkit/gc-explorer/) y más.

**Lo que encontrará:**
- Documentación de referencia para el uso de GC Scripts Hub, GC Explorer, Apache Superset, Filebrowser y otras herramientas
- Cómo utilizamos servicios externos para proporcionar funcionalidad adicional a Guardian Connector

**Ideal para:** Comprender la plataforma Guardian Connector, planificar flujos de trabajo y aprender cómo diferentes herramientas trabajan juntas en el ecosistema de Guardian Connector.

### 🔗 **[Aplicaciones conectadas](./connected-applications/)**

Documentación para herramientas de recopilación de datos y plataformas externas con las que Guardian Connector puede integrarse como un centro de datos centralizado.

**Lo que encontrará:**
- Herramientas principales de recopilación de datos (KoboToolbox, CoMapeo, Timelapse) compatibles con Guardian Connector 
- Plataformas externas con las que Guardian Connector puede integrarse como un centro de datos centralizado

**Ideal para:** Comprender los flujos de trabajo de recopilación de datos, planificar integraciones y aprender cómo Guardian Connector agrega datos de múltiples fuentes.

### 🌐 **[Aplicaciones recomendadas](./recommended-applications/)**

Documentación para aplicaciones recomendadas que son útiles para tareas y flujos de trabajo comunes.

**Lo que encontrará:**
- Orientación sobre el uso de herramientas que hemos encontrado útiles, como [QGIS](/reference/recommended-applications/qgis) y [OpenRefine](/reference/recommended-applications/openrefine)

### 📚 **[Flujos de trabajo comunes](./common-workflows/)**

Orientación para flujos de trabajo comunes al usar Guardian Connector.

**Lo que encontrará:**
- Orientación sobre cómo utilizar sus datos en [QGIS](/reference/recommended-applications/qgis) u otro software SIG
- Información sobre cómo descargar todos sus datos

### 🌐 **[Alojamiento y costos](./hosting/)**

Orientación técnica para implementar y alojar la infraestructura de Guardian Connector para cumplir con los requisitos de soberanía de datos de su comunidad.

**Lo que encontrará:**
 
- Opciones y requisitos de alojamiento
- Consideraciones sobre la infraestructura en la nube
- Estimaciones de costos y recursos de planificación
- Consideraciones sobre seguridad y soberanía de datos
- Procedimientos de mantenimiento y actualización

**Ideal para:** Comprender cómo implementar y alojar la infraestructura de Guardian Connector para cumplir con los requisitos de soberanía de datos de su comunidad.

### 🔧 **[Para desarrolladores](./for-developers/)**

Información para desarrolladores interesados en contribuir a Guardian Connector o en crear integraciones personalizadas.

**Lo que encontrará:**
- Enlaces a los repositorios principales
- Configuración del entorno de desarrollo
- Instrucciones de despliegue y alojamiento
- Pautas de contribución
