# 📚 Programa de estudos: Gerenciamento de Dados com KoboToolbox

## **Visão geral**

Este programa de aprendizado combinado, com ritmo próprio e duração de 4 semanas, combina tutoriais em vídeo, materiais de leitura e exercícios práticos.

Serão agendadas chamadas de vídeo semanais para responder a perguntas e fornecer suporte.

Ao final deste programa de aprendizagem, você terá adquirido o conhecimento necessário para gerenciar todo o fluxo de trabalho do KoboToolbox.

## Guia do instrutor

Caso você queira organizar um workshop como este, encontrará informações muito valiosas no nosso [Manual do Instrutor](./instructors-guide.md).
