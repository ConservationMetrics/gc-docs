# Aplicações recomendadas

Dentro do ecossistema de dados, existem diversas ferramentas disponíveis para uma variedade de tarefas. Ao longo da nossa própria jornada com dados, identificamos várias que se destacam como particularmente úteis para tarefas e fluxos de trabalho comuns.

- [QGIS] (/referência/aplicações-recomendadas/qgis) – Uma aplicação de código aberto para trabalhar com dados geográficos.
- [OpenRefine] (em /referência/aplicações-recomendadas/openrefine) – Uma ferramenta para limpar e transformar conjuntos de dados.
